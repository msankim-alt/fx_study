📋 API 키 발급 안내

  1. console.groq.com/keys 접속
  2. 회원가입(무료) 후 "Create API Key" 클릭
  3. 생성된 키(gsk_...) 복사
  4. 위 입력창에 붙여넣기 후 "저장"
  5. 무료 플랜으로 llama-3.3-70b 모델 사용 가능
